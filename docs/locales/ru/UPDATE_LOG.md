# Журнал обновлений

## 2026-04-10

### Добавлено

- Распознавание входящих изображений в каналах: Telegram / Feishu / личный WeChat сохраняют распознаваемые картинки в workspace и передают в выполнение через `mentions(file)`.
- Команда прерывания для каналов: `/wtf`, единая ветка в `TaskAPI.executeTaskStream`.

### Изменения поведения

- `/wtf` выровнен с Web ESC: прерывается только текущая задача сессии, очередь не очищается.
- `/wtf` обрабатывается до проверки busy, чтобы команда не превращалась в обычный «занято».

### Проверка

- Регрессия: `task-api-execute-stream-slash`, `telegram-squid-bridge`, `feishu-squid-bridge`, `weixin-personal-squid-bridge`.
